simple
======
仿制纯白小清新模板
For 贴吧签到助手(by kookxiang)

使用方法：<br />
1.将下载的simple-master.zip解压<br />
2.将解压到的文件夹重命名为simple（重要）<br />
3.上传文件夹到贴吧签到助手目录下template文件夹内<br />
4.到管理面板-模板管理中启用纯白小清新（仿制）<br />

提醒：<br />
使用该版需要您是贴吧签到助手的管理员<br />
